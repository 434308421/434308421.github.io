JetBrains IntelliJ IDEA是使用Java语言的智能、功能强大且集成的软件开发工具。但是，此IDE不仅限于Java支持，还支持从Web设计语言到数据库和移动应用程序的各种编程语言和框架。该程序的高智能性为程序编写者提高编码的速度和准确性提供了极大的帮助。

功能概述

  • 检测代码并帮助智能完成高级分析
  • 能够在集成环境中编辑和开发不同的编程语言代码
  • 支持最基本和最重要的Web标准和框架，例如Spring MVC，GWT，Vaadin，Grails，JSF，Struts，Flex和…以及Web语言，例如HTML，CSS，SASS，LESS，JavaScript，CoffeeScript，Node.js，ActionScript
  • 包括用于编辑各种数据库（例如PostgreSQL，MySQL，Oracle，SQL Server）的完整工具
  • 在Android平台下开发移动应用程序，包括用户界面设计和程序优化
  • 支持适用于Android和iOS的Adobe AIR技术
  • 具有版本控制工具
  • 具有各种插件以增强功能并优化编程环境

安装要求

  • Microsoft Windows 10、8的64位版本
  • GNOME或KDE桌面
  • 最低2 GB RAM，建议8 GB RAM
  • 2.5 GB硬盘空间，建议使用SSD
  • 最低1024×768屏幕分辨率

软件安装

1.右键以管理员身份运行

2.点击next

3.修改或默认安装目录，点击next

4.根据系统及需要勾选，点击next

5.点击install

6.开始安装，耐心等待

7.勾选取消运行，点击finish

8.重新打开软件

9.跳过设置

10.点击evaluate for free ---evaluate

11.打开crack文件夹

打开jetbrains-agent-latest---jetbrains-agent---lib文件夹

将jetbrains-agent.jar文件拖进对话框，弹出提示，点击restart

12.选择activation code 点击为IDEA安装

13.点击是

14.查看激活状态

点击configure---manage license

订阅到2089年到期

15.修改界面语言

2020.1版本自身带有中文版插件，安装后即可使用中文版

再次点击configure---plugins

搜索框输入Chinese，点击install，完成后点击restart即可

界面变为中文

面板部分基本全部汉化
